# Missão Aurora Siger - Sistema de Pré-Decolagem

Este projeto faz parte da atividade integradora para o desenvolvimento de um **Relatório Operacional de Pré-Decolagem**. O objetivo é simular o sistema de decisão de uma nave espacial utilizando telemetria e automação em Python.

## Explicação do Projeto
O sistema recebe dados de sensores (temperatura, energia, integridade e pressão) e utiliza um algoritmo lógico para decidir se o lançamento deve ser **AUTORIZADO** ou **ABORTADO** com base em faixas de segurança predefinidas.
